# Toggle Edge Scrolling mod for Cities: Skylines 2

Quickly turn the camera's mouse edge scrolling on or off - no more digging through the options menu every time.

## Use Case
If you're running a multi-monitor setup or with a small screen, you may find that the mouse edge scrolling is sometimes annoying. Even more annoying is when you alt-tab to check your email or order pizza and come back to find the camera has moved to the edge of the map.

## Features
- Inspired by algernon's [Toggle Edge Scrolling mod](https://steamcommunity.com/sharedfiles/filedetails/?id=2645254941) for Cities: Skylines.
- Toggle edge scrolling with a configurable hotkey (default Ctrl + Shift + E).
- Optional on-screen toggle button, shown either in the main UI (top-right of the screen) or in the game's mod button menu.
- The button is green when edge scrolling is on and red when it is off.
- Optionally disable edge scrolling automatically on startup.
- Optionally disable edge scrolling while the game window is in the background, then restore it when the window is focused again.

## Optional
- Unified Icon Library - if installed, the toggle button uses a matching game-style icon. It is not required; without it the mod uses its own bundled icon.

## Usage
Configure the hotkey and button placement under Options &gt; Toggle Edge Scrolling.
